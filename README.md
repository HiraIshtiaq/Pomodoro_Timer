# 🍅 Pomodoro Timer

A beautifully crafted Pomodoro Timer that helps you stay focused with 25-minute work sessions followed by 5-minute breaks. Track your daily progress, customize durations, and enjoy satisfying audio cues.

## ✨ Features

- ⏰ **Smart Timer** - Start, pause, reset, or skip sessions
- 🎨 **Visual Progress Ring** - 60 tick marks fill as time elapses
- 📊 **Daily History** - Automatically tracks completed focus sessions
- 🎵 **Audio Feedback** - Gentle chimes for session completions, soft ticks for final 10 seconds
- ⌨️ **Keyboard Shortcuts** - Space/Enter (start/pause), R (reset), S (skip)
- 📱 **Fully Responsive** - Works perfectly on phones, tablets, and desktops
- 💾 **Persistent Storage** - Settings and history survive page reloads
